# 📊 Instagram Engagement Analysis Dashboard

## 📌 Project Overview

This project analyzes Instagram post performance to uncover insights on audience engagement, content effectiveness, and growth trends using Power BI.

## 📂 Dataset Features

- Post ID, Date, Type

- Caption Length, Hashtags

- Likes, Comments, Shares, Saves

- Reach, Impressions

- Follower Count

## 🛠️ Data Cleaning

- Removed duplicates and null values

- Converted date format

- Created Month-Year column for trend analysis

## 📐 Key Metrics (DAX)

- Engagement Rate

- Interaction-to-Follower Ratio

- Total Engagement


## 📊 Dashboard Features

- Trend Analysis (Reach, Impressions, Engagement Rate)

- Top 5 Performing Posts

- Post Type Comparison

- Follower Growth Impact

- Interactive Filters (Post Type, Month-Year)

## 🔍 Key Insights

- Reels have the highest engagement rate

- Optimal hashtag usage improves performance

- Engagement varies by month (seasonality observed)

- Follower growth positively impacts engagement

## 🚀 Tools Used

- Power BI

- DAX

- Excel
