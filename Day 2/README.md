# 50days-Coding-Challenge

**Day-2**

**Excel Data Analyst Challenge – Educational Data Analysis**

📌 **Project Overview**

This project is part of a Data Analyst practice challenge where students data is analyzed using Microsoft Excel.
The objective of the challenge is to perform data exploration, calculate average test scores, create pivot tables, and derive meaningful  insights from the dataset.
The dataset contains 200 students data including age, study hours, screen time, test scores and extra curricular hours .

📂 **Dataset**

The dataset contains the following fields:

Column   -->  	Description

Student_ID  -->	Unique student identifier

Age  --> Age of the student

Study hours  --> hours of study 

Screen time	-->  Student mobile exposure

Test scores   -->  scores  gained by students

Extra curricular hours  --> Number of hours student has used for other activities

🧹 **Data Cleaning**

Before performing the analysis, the dataset was reviewed to ensure it was clean and ready for analysis.

Steps performed:
•Verified column data types.

•Checked for missing values or null values and replaced with 0.

•Ensured numeric formatting for all columns.

•Created a calculated columns.

📊 **Data Analysis**

Using Pivot Tables in Excel, the dataset was analyzed to answer questions.

Analysis Performed-Pivot tables

•Screen time vs test scores

•Screen time vs extra curricular hours

•Age wise performance

•Inserted slicers.

📈 **Visualizations**

Charts were created in Excel to better understand the students performance.

Visualizations include:

•Screen time vs test scores (Scatter plot)

•Average test scores by screen time (Bar graph)

🔎 **Key Insights**

•Despite high screen time, the average test scores are high as may be study hours impact their scores.

•As the data has given high screen time, students have more extra curricular hours.

•Helps identify which age group scores best.
