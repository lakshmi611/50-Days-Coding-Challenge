# Medical Insurance Cost Analysis (SQL Project)

## 📌 Objective

Analyze insurance data to identify cost patterns based on age, smoking habits, BMI, and region.

## 🛠️ Tools Used

- MySQL

## 📊 Key Analysis

- Total records count

- Smokers vs Non-smokers comparison

- Gender distribution

- Regional cost analysis

- Age group segmentation

- BMI impact on insurance

## 🔍 Key Insights

- Smokers pay significantly higher insurance

- Northwest region has highest average cost

- BMI and smoking together drastically increase cost

## 📁 Files Included

- SQL Queries

- Output Results

## 🚀 Conclusion

Insurance cost is highly influenced by smoking habits, age, and BMI.
