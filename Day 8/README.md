# 📦Logistics & Supply Chain Dashboard (Power BI)

## 📌 Objective

Analyze delivery performance, identify delays, and improve cost efficiency.

## 📊 KPIs

*Total Shipments

*On-time Delivery %

*Avg Delivery Time

*Total Delivery Cost

## 📈 Dashboard Features

*Heatmap: Delivery Status by City

*Bar Chart: Avg Delivery Time by Region

*KPI Cards

## 🔍 Key Insights

*Some regions show higher delays

*On-time performance needs improvement

*High delivery cost linked with delays

## 🚀 Recommendations

*Optimize logistics routes

*Focus on delay-prone cities

*Improve delivery planning
