 ## 📊 Credit Risk Analysis & Loan Portfolio Management

📌 **Project Overview**
This project focuses on analyzing a bank’s loan portfolio to assess credit risk, identify trends, and provide strategic recommendations for better lending decisions. The analysis was performed using Excel, including data cleaning, KPI calculations, pivot tables, and What-If analysis.

 🎯 **Objectives**
- Evaluate the overall health of the loan portfolio  
- Identify high-risk segments and default patterns  
- Analyze lending trends over time  
- Perform scenario-based financial analysis  
- Provide actionable recommendations for risk management  


## 📂 Dataset Description
The dataset includes the following key attributes:
- Loan ID, Borrower ID  
- Loan Amount, Interest Rate, Term  
- Loan Status (Fully Paid, Current, Charged Off)  
- Loan Grade (A–D)  
- Loan Intent  
- Annual Income, Home Ownership  
- Debt-to-Income (DTI), Loan-to-Value (LTV)  
- Total Payments Received, Recovery Amount  


 🧹**Data Cleaning**
- Removed duplicate records  
- Standardized date and categorical formats  
- Handled missing values  
- Ensured correct data types for all columns  


📊 **Key Performance Indicators (KPIs)**
- **Loan-to-Value (LTV)** = Loan Amount / Property Value  
- **Debt-to-Income (DTI)** = Total Debt / Annual Income  
- **Months on Book** = Loan Age  
- **Profit/Loss** = Total Payments Received – Loan Amount  


📈 **Data Analysis & Visualizations**

🔹** Pivot Table Analysis**
- Loan distribution by **Loan Status**  
- Loan distribution by **Loan Grade**  
- Loan segmentation by **Loan Intent**  

🔹**Visualizations**
- 📉 Monthly loan issuance trends (Line Chart)  
- 📊 Loan grade distribution (Bar/Pie Chart)  

🔍 **Key Insights**
- Majority of loans fall under **low-risk grades (A & B)**  
- Higher default rates observed in **Grade C/D loans**  
- Borrowers with high **DTI ratios** show increased risk  
- Certain regions exhibit higher default concentration  
- Debt consolidation is the most common loan purpose  


✅ **Recommendations**
- Implement stricter checks for high DTI borrowers  
- Focus lending on low-risk segments (Grade A & B)  
- Use risk-based interest rate pricing  
- Monitor high-risk regions closely  
- Adjust lending strategy based on economic scenarios  


🛠**Tools Used**
- Microsoft Excel  
  - Pivot Tables  
  - Charts  

📌 **Conclusion**
The analysis highlights that while the portfolio is largely stable, specific borrower segments and regions pose higher risks. Strategic adjustments in credit policies and interest rates can significantly improve profitability and reduce default risk.
