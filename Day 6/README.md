# 📊 Banking Sector Analysis Dashboard

## 📌 Project Overview

This project analyzes customer behavior, transactions, and loan performance using Power BI. The goal is to generate actionable insights for better decision-making.

## 🧹 Data Cleaning & Transformation

- Removed duplicates and handled missing values  

- Created Net Transaction Impact column  

- Grouped customers into Age Groups  

- Standardized Account Types  

## 🔗 Data Modeling

- Created Customer, Transaction, Loan, and Calendar tables  

- Established relationships using Customer ID  

- Implemented star schema  


## 📊 Key DAX Measures

- Total Deposits  

- Total Withdrawals  

- Net Balance Growth  

- Loan Default Rate %  

- Customer Profitability Score  

## 📈 Dashboard Pages

1. Customer Overview  

2. Transaction Analysis  

3. Loan Performance  

## 💡 Key Insights

- Young customer base dominates  

- Stable transaction growth with peak periods  

- Moderate loan default risk  

- Top customers contribute major revenue  


## 🚀 Recommendations

- Focus on high-value customers  

- Improve engagement of low-balance customers  

- Monitor high-risk loans  

- Optimize marketing during peak hours
