# 50days-Coding-Challenge

**Day-3**

**Excel Data Analyst Challenge – Nutrition & Growth Analysis**

📌 **Project Overview**

This project is part of a Data Analyst practice challenge where childrens data is analyzed using Microsoft Excel.
The objective of the challenge is to perform data exploration, calculate average test scores, create pivot tables, and derive meaningful  insights from the dataset.
The dataset contains 100 children data including age, gender,height,weight,food intake, screen time & region

📂 **Dataset**

The dataset contains the following fields:

Column   -->  	Description

Child ID  -->  Unique identifier for each child. 

Age_Months  --> The child's age in months. 

Gender  -- > The child's gender (Male/Female). 

Height_cm  --> The child's height in centimeters. 

Weight_kg --> The child's weight in kilograms. 

Daily_Fruit_Servings  --> Average number of fruit servings per day. 

Daily_Veggie_Servings  --> Average number of vegetable servings per day. 

Daily_Sugar_Drinks  --> Average number of sugary drinks per day. 

ScreenTime_Hours  -->  Average daily screen time in hours. 

Region    --> The geographic region where the child lives. 

🧹 **Data Cleaning**

Before performing the analysis, the raw dataset was load into excel and reviewed to ensure it was clean and ready for analysis.

Steps performed:

•Verified column data types.

•Checked for missing values or null values.

•Ensured numeric formatting for all columns.

•Created calculated columns.

📊 **Data Analysis**

Using Pivot Tables in Excel, the dataset was analyzed to answer questions.

Analysis Performed-Pivot tables

•Health status by age group

•Nutrient intake vs BMI

•Correlation of sugar & screen time

•Regional differences


📈 **Advanced analysis & Visualizations**

Charts were created in Excel to better understand the students performance.

Visualizations include:

•Health status by age group (Clustered column chart)

•Nutrient intake vs BMI(column chart)

•Correlation of sugar & screen time(Clustered bar chart)

•Regional differences (3D area chart)

•Inserted slicers.

•Inserted data validation rule.

🔎 **Key Insights**

•Children with higher fruit and vegetable intake tend to have a healthier BMI, indicating a strong positive relationship between balanced diets and proper growth.							

•Increased consumption of sugary drinks is strongly associated with higher BMI levels, particularly in overweight and obese categories.					
					
•Screen time shows a positive correlation with poor health outcomes — children with higher screen hours tend to fall into overweight or obese groups.							

•Regional differences highlight that some regions have higher average BMI and lower fruit intake, suggesting unequal access to healthy food or lifestyle awareness.										

•Younger age groups generally show better health distribution, while older children display a gradual increase in overweight cases.
