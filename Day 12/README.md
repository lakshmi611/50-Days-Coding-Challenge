# 📊 Payment Efficiency Analysis Dashboard

## 📌 Project Overview
This project analyzes payment generation efficiency based on operational data, focusing on the percentage of payments generated within 15 days.


## 📂 Dataset

- Financial Year

- Payment Efficiency (%)

- Operational indicators


## 🧹 Data Cleaning

- Removed null values

- Standardized column names

- Converted percentage fields to numeric format


## 📊 Key KPIs

- Average Payment %

- Maximum Performance

- Minimum Performance

- Total Records


## 📈 Dashboard Features
- KPI Cards for quick insights

- Performance Distribution (Donut Chart)

- Detailed Table

## 🔍 Insights

- Majority of entries exceed 100%, indicating strong performance or data inconsistencies

- Some segments show lower efficiency, requiring attention

## 💡 Recommendations

- Improve data accuracy and reporting standards

- Focus on low-performing segments

- Implement consistent monitoring mechanisms
